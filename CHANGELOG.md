# Changelog

All notable changes to this project are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and releases use semantic versioning.

## [Unreleased]

### Added

- Search-focused documentation and analytical problem index.
- Python data-visualisation guide covering static, interactive, geospatial, dashboard, large-data, accessibility, and reproducibility practices.
- PyPI Trusted Publishing workflow.
- Dependabot configuration for Python and GitHub Actions dependencies.
- CodeQL security scanning.
- Security and support policies.
- Structured bug-report, feature-request, and pull-request templates.
- Pre-commit checks for Ruff, YAML, whitespace, merge conflicts, and oversized files.

### Changed

- Expanded package metadata, classifiers, project URLs, and search keywords.
- Improved README navigation, examples, use cases, and discoverability.

## [1.0.0] - 2026-07-19

### Added

- Reusable static and interactive visualisation APIs.
- Matplotlib, Seaborn, Plotly, Streamlit, geospatial, and Datashader examples.
- Deterministic chart gallery and rendering benchmark.
- Automated tests, branch coverage, linting, and continuous integration.
- Citation metadata and documentation linking published articles and presentations.
